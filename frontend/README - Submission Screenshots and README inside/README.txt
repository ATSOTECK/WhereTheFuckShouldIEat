Functionality:

trans-for-Market: returns all transactions for Market but 
		  not ordered account by account or showing account name
one-account: Works to Spec
trans-for-route on routeID: Works to Spec
all-routes: Works to Spec

How It Works:

	By Page: 
		Transactions - returns number of transactions
		Number of Transactions - returns number of transactions
		Single Account Transactions - returns transactions for inputted single account
		Route ID Transactions - returns transactions for inputted routeID
		All Routes Transactions - returns all routes for transactions cycle
		Market Transactions - returns market transactions for inputted market
